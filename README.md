# Spree Starter

A Rails application template pre-configured with [Spree Commerce](https://spreecommerce.org).

## Installation

If you want a full project scaffold (this backend + optional Next.js storefront + the `spree` CLI), use [`create-spree-app`](https://spreecommerce.org/docs/developer/cli/quickstart):

```bash
npx create-spree-app my-store
```

Just evaluating Spree? Skip self-hosting entirely and use a [hosted sandbox](https://spreecommerce.org).
